Cake Decorating Game

Cake Decorating is a game where the user has to match the cake to the best of their ability. It has two levels, level 1 and level 2. As well as a free draw level. In level one, the user is given a colored cake and an outlined cake. The user must use the color palette provided to them, and match the colors from the already filled in cake. In level 2, the user is given 15 seconds to look at a cake. After the 15 seconds, the cake disappears, and the user has to match the colors of the cake with the outline using the color palette. In the free draw level, the user can get creative and color the cakes in whichever way the wish. Level 1 and 2 have a check button that makes sure the user uses the right colors of the cake.
